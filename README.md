# Py-Choco
Choco API for python

## Get started 
Set your JDK_HOME

```shell
 set JDK_HOME=YOUR_JDK_PATH
```

Add JVM to your PATH

```shell
 set PATH=YOUR_OLD_PATH;C:\...\YOUR_JDK_HOME\jre\bin\server
```
